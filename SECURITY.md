# Security

RunHold is security-sensitive because it monitors keyboard state and supported mouse trigger buttons, and it injects keyboard input.

RunHold does not include networking, telemetry, credential capture, macro recording, or raw typed-text logging.

To report a security issue, use the GitHub repository issue or contact path:

https://github.com/hfunball/RunHold

Please include the app version, Windows version, install method, and enough detail to reproduce the behavior.
