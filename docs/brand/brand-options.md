# Brand Options

## KeyLatch

Clear, functional, and the strongest descriptive name. It says exactly what the app does.

- Dark palette: graphite, near-black, electric blue, cool gray.
- Light palette: white, blue-tinted gray, electric blue, charcoal.
- Tray mark: a keycap with a small latch notch.
- Tone: precise, low-noise, practical.

## KeyRun

Good gaming energy and a useful double meaning: running in a game and running a repeated process. It is less ownable because search results already show old utilities, CLI tooling, and other projects using the name.

- Dark palette: blackened navy, bright cyan, steel blue, soft white.
- Light palette: white, pale blue, cyan, slate.
- Tray mark: a forward keycap with a subtle motion line.
- Tone: faster, more game-adjacent, still understated.

## PressLatch

Practical and mechanical. Slightly less friendly than KeyLatch, but accurate.

- Dark palette: carbon, signal green, zinc, white.
- Light palette: white, pale green, charcoal, zinc.
- Tray mark: a switch toggled into a locked state.
- Tone: technical, direct, utility-first.
